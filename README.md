# 🍕 Pizza Hut Sales Analysis using SQL

## 📌 Project Overview

This project analyzes the Pizza Hut sales dataset using SQL to generate meaningful business insights. It focuses on revenue analysis, customer ordering patterns, best-selling pizzas, and sales trends.
